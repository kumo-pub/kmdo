# kmdo pkg

Release engineering, simplified

## Synopsis

Release engineering, simplified.

Kmdo is a release automation tool, built with love and care by @caarlos0 and many contributors.

Complete documentation is available at https://kmdo pkg.com

## Examples

```

# Initialize your project:
kmdo pkg init

# Verify your configuration:
kmdo pkg check

# Verify dependencies:
kmdo pkg healthcheck

# Build the binaries only:
kmdo pkg build

# Run a snapshot release:
kmdo pkg release --snapshot

# Run a complete release:
kmdo pkg release
		
```

## Options

```
  -h, --help      help for kmdo pkg
      --verbose   Enable verbose mode
```
