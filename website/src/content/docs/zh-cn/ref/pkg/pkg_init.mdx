# kmdo pkg init

Generates a .kmdopkg.yaml file

```
kmdo pkg init [flags]
```

## 选项

```
  -f, --config string     Load configuration from file (default ".kmdopkg.yaml")
  -h, --help              help for init
  -l, --language string   Which language will be used (default "go")
```

## 从父命令继承的选项

```
      --verbose   Enable verbose mode
```
