Kmdo 支持多种编程语言和构建系统，通过其 _builder_ 接口实现。

每个 _builder_ 接收构建配置，并将输出的二进制文件或库写入 `dist` 目录。

目前支持多种 builder，并将持续增加更多选项。

以下是支持的构建方式列表：
