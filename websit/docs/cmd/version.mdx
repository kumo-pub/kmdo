# kmdo version 命令文档

## 概述
kmdo version 命令用于输出 kmdo 工具的版本信息及 Git 提交 ID，快速查看当前使用的工具版本和构建关联的代码提交记录。

## 命令结构
kmdo version

## 核心功能
输出 kmdo 工具的版本标签（VERSION）和 Git 提交 ID（GIT ID），无额外子命令或参数。

## 详细命令说明
### 功能
展示当前 kmdo 二进制文件的构建版本信息，包括版本标签和对应的 Git 提交 ID，帮助确认工具版本及代码构建来源。

### 用法
kmdo version

### 输出说明
执行命令后将直接输出两行信息，格式固定：
VERSION: [构建版本标签]
GIT ID: [构建对应的Git提交ID]

### 示例
# 查看 kmdo 版本信息
kmdo version

# 输出示例
VERSION: v1.0.0
GIT ID: a1b2c3d4e5f67890abcdef1234567890abcdef12

## 信息说明
| 输出字段 | 含义 |
|----------|------|
| VERSION | 工具的构建版本标签，通常遵循语义化版本规范（如 vX.Y.Z），由环境变量 environment.BuildTag 提供 |
| GIT ID | 构建当前二进制文件时对应的 Git 提交完整 ID，唯一标识代码版本，由环境变量 environment.BuildGitID 提供 |

## 注意事项
1. 该命令无任何 Flags 或参数，执行后直接输出版本信息，无需额外配置。
2. 版本信息来源于工具构建时注入的环境变量（BuildTag 和 BuildGitID），非运行时动态获取。
3. 若 BuildTag 或 BuildGitID 未在构建时正确设置，对应字段可能显示为空或默认值。
